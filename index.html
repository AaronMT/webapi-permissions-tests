B2G Apps Tests

Hosted App Install - No Appcache Preload

Note: Only one of these hosted apps is allowed to be installed at one time.

Hosted App Test Case 1

App has valid launch path, an icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name and url specified.

Hosted App Test Case 2

App has valid launch path, no icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name and url specified.

Hosted App Test Case 3

App has valid launch path, icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name and url specified. App name, description, and developer name all have Japanese characters and English characters in the fields.

Hosted App Test Case 4

App has valid launch path, icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name specified.

Hosted App Test Case 5

App has valid launch path, icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer url specified.

Hosted App Test Case 6

App has valid launch path, icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name and url specified. The app name and developer name and url are overridden to a localized value if the phone's locale is English. Only one locale override is included.

Hosted App Test Case 7

App has valid launch path, icon specified, web type, geolocation, desktop notification, and fmradio permissions called out, and a developer name and url specified. The app name and developer name and url are overridden to a localized value if the phone's locale is English or Portuguese. Two locale overrides are included.

Hosted App Install - Appcache Preloading

Hosted App with Appcache Preloading Test Case 1

App is a hosted app that has a valid launch path, developer name and url, an icon, and a valid small appcache preloaded upon installation of the app.

Packaged App Install

Packaged App Test Case 1

App is a privileged packaged app that is not signed. Mini-manifest estimates size to be 16 KB. Has a name, valid launch path, icon, device storage pictures permission, contacts permission, and developer name and url.

Packaged App Test Case 2

App is a web packaged app that is not signed. Mini-manifest estimates size to be 10 KB. Has a name, valid launch path, icon, device storage pictures/music/sdcard/videos permissions, contacts permissions, geolocation permissions, desktop notification permissions, fmradio permissions and a developer name and url.

Packaged App Test Case 3

App is a web packaged app that is not signed. Mini-manifest estimates size to be larger than the space on the phone. Has a name, valid launch path, icon, device storage pictures/music/sdcard/videos permissions, contacts permissions, geolocation permissions, desktop notification permissions, fmradio permissions and a developer name and url.
